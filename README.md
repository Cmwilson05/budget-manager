# Budget Manager

A personal finance workbench built with React, Vite, Tailwind CSS, and Supabase.

## Features
*   **Accounts**: Track assets and liabilities.
*   **Workbench**: Forecast cash flow with toggleable transactions.
*   **Bill Library**: Templates for recurring expenses.
*   **Notes**: Rich text shared notepad.
*   **Multi-Workbench**: Separate planning for Main Cash Flow and Credit Cards.

## Deployment
Deployed to GitHub Pages via GitHub Actions.
Secrets required:
*   `VITE_SUPABASE_URL`
*   `VITE_SUPABASE_ANON_KEY`
